# Incident Management System – ServiceNow Project

## Overview
This project demonstrates the implementation of an Incident Management workflow using ServiceNow ITSM concepts.

## Features
- Incident ticket creation
- Automatic priority assignment
- Automated team assignment
- Email notifications
- Incident lifecycle management
